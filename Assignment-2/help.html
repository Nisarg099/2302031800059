Frequently Asked Questions

How do I create or join a room?

To create a room, go to the Rooms page and click "Create Room". To join, either use a room code or select "Join Room" from the available rooms list.

What happens if I lose connection during a game?

Don't worry! The game automatically saves your progress. You can rejoin the room using the same room code within 30 minutes.

How do hints work?

Each room comes with 3 hints. Using a hint will provide a clue about your current objective. Note that using hints affects your final score.
